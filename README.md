
# XO Game 🎮

A Tic-Tac-Toe game built with Flutter.

## Features
- Two player mode (X and O)
- Score tracking
- Restart game button

## Built With
- Flutter
- Dart

## Getting Started
1. Clone the repo
2. Run `flutter pub get`
3. Run `flutter run`
